# Contributing

UMBRAL welcomes code, translations, protocols, null results, replications, and safety critiques. Open an issue first for experiments involving people. Keep claims narrower than evidence, add a sham/control, state failure criteria, and never commit participant data or secrets.

Development: install dependencies, run `npm run lint`, `npm run typecheck`, `npm test`, and `npm run build`. By contributing code you agree it is available under AGPL-3.0.
